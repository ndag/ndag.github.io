Formigramator: Formigram calculation tool Developed by Dave Verano and Woojin Kim

Given a time-evoloving network (i.e. dynamic network), the evolution of connected components can be summarized into the so-called formigram . Formigrams can be viewed as endriched Reeb graphs , a popular tool in topological data analysis. The concept of formigrams is so intuitive that one can grasp the essential idea from the figure below.

Input for Formigramator

Here is an example of the input dynamic network data:
{{1,1},{2,2},{3,3},{4,4}}
t1 C {1,2}
t2 A {5}
t3 B {1,2}
t4 R {1}
EOF
To interpret, the input dynamic network begins with 4 vertices, named 1,2,3,4 (Line 1). At time t1, vertices 1 and 2 are Connected by an edge (Line 2). At time t2, a new vertex 5 is Added. At time t3, the connection {1,2} Breaks (Line 3). At time t4, the vertex 1 is Removed (Line 4). The input ends with EOF, the abbreviation for "End of file".

User guide of Formigramator Written by Woojin Kim

  1. Go to https://editor.p5js.org/ and follow the instruction below.
  2. Delete the default filler in the editor. Then, copy and paste the code in formigramator.txt into the editor.
  3. Click "Sketch" at the top and click "Add File". Then, input "edges.txt".
  4. Click ">" right below the play button and click "edges.txt"
  5. Copy and paste the code below into the editor for "edges.txt".
    {{1,1},{2,2},{3,3},{4,4}}
    t1 C {1,2}
    t2 A {5}
    t3 B {1,2}
    t4 R {1}
    EOF
  6. Click the play button and then you can see the output formigram in the preview window.
  7. You can compute your own dynamic network by modifying "edges.txt", and tuning the parameters as below.

    8. Refer to [1],[2] for the smoothing parameter for formigrams.

Applications: Clustering feature analysis of dynamic metric data

Given a dynamic point cloud, fix a certain connectivity parameter δ>0. At each time, any two points within distance δ in the point cloud are connected by an edge; this yields a dynamic network. We summarize/visualize the evolution of connected components of this dynamic network into the formigram.
